# Security Policy

Aegis Code is a harness for agentic coding work, so security reports may cover
prompt/control bypasses, sandbox escapes, command mediation failures, provider
leakage, evidence tampering, or unsafe context-pack promotion.

## Reporting

Report security issues privately to the maintainers. Do not open a public issue
for exploitable behavior until a maintainer has confirmed disclosure timing.

## Sensitive Data

Do not include secrets, tokens, private prompts, or customer data in reports.
Redact command output and logs wherever possible.
